# Fake News Detection using LSTM

This project uses **Deep Learning and NLP** to detect whether a news article is **Real** or **Fake**.

Built with:

* Python
* TensorFlow / Keras
* LSTM
* Streamlit

## 📌 What This Project Does

* Takes news text as input
* Processes the text using NLP
* Predicts whether the news is:

  * ✅ Real
  * ❌ Fake

---

## 🧠 Model Used

This project uses:

* Tokenization
* Padding
* Embedding Layer
* LSTM
* Dense Layers
